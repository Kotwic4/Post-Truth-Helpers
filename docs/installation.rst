============
Installation
============


Stable release
--------------

To install Post Truth Helpers, run this command in your terminal:

.. code-block:: console

    $ pip install post_truth_helpers

This is the preferred method to install Post Truth Helpers, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Post Truth Helpers can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Kotwic4/post_truth_helpers

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/Kotwic4/post_truth_helpers/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Kotwic4/post_truth_helpers
.. _tarball: https://github.com/Kotwic4/post_truth_helpers/tarball/master
